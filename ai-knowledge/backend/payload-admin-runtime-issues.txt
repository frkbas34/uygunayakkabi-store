## Resolved Issues (History)

### Issue: PayloadComponent not found in importMap
- **Status**: RESOLVED
- **Cause**: Auto-generation of importMap.ts failed due to Windows/Linux esbuild mismatch
- **Fix**: Manual reconstruction of importMap.ts with 30+ Payload components
- **Constraint**: importMap.ts must be maintained manually (D-024 related)

### Issue: Admin panel unstyled / CSS not loading
- **Status**: RESOLVED
- **Cause**: Missing CSS import in admin layout
- **Fix**: `@payloadcms/next/css` imported in (payload)/layout.tsx

### Issue: Admin panel not in Turkish
- **Status**: RESOLVED
- **Fix**: Added `import { tr } from "@payloadcms/translations/languages/tr"` + `i18n` config in payload.config.ts
