Key Decisions (D-070 — D-073)
D-070
11-Skill Stack with 3-Level Rollout
Safe evolution from single-skill bot to full operations assistant. Level A active, Level B controlled, Level C observe-only.
D-071
agent-memory over ChromaDB
File-based memory aligns with existing docs, avoids extra Docker container, easier to inspect/debug. Can upgrade to ChromaDB later.
D-072
Learning Engine: Observe-First
No auto-modification of production behavior. Must build trust through accurate observations before earning autonomous rights (30+ days).
D-073
Draft-First Publishing, Confirm Writes
Publishing, DB writes, and image ops all gated. High-impact, hard-to-reverse mistakes prevented by confirmation gates.
🚀 Deployment Steps (VPS Operator Actions)
Copy skills to VPS: scp -r mentix-skills/* furkan@VPS:/home/furkan/.openclaw/skills/
Create memory dirs: mkdir -p /home/furkan/.openclaw/skills/agent-memory/data/{incidents,patterns,knowledge,decisions,rewards}
Add env vars to OpenClaw Docker: DATABASE_URI and GITHUB_TOKEN
Restart gateway: cd /opt/openclaw && docker compose restart
Verify skill recognition in logs: docker logs openclaw-openclaw-gateway-1 --tail 50
Test Level A skills via Telegram (one by one)
