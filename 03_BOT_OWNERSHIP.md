# Bot Ownership

Last updated: 2026-06-21

## Mentix / Uygunops

Main operator bot for product intake, stock, image generation, diagnostics, publishing helpers, leads, orders, funnel, UTM, and campaign helpers.

## GeoBot

Content and GEO/SEO bot for product descriptions, audits, product intelligence, preview, and publish support.

## OpenClaw

Agent brain and skill host. Use it for reasoning, diagnostics, memory, drafting, and operator support.

## Next/Payload Telegram Route

Operational API layer that receives bot events and executes app workflows.

## n8n

Optional workflow glue. Do not expand n8n until there is a clear current need.

## SupplierScout

Dormant. Do not activate unless the user explicitly reverses the own-products-only decision.
